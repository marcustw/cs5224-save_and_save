# Save & Save

## Installation

_Save & Save_ requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
npm install
npm start
```

## Pages routes

- /dashboard
- /cart
- /store/items
- /search/items
- /user/account
